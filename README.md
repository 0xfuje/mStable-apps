## Development

This project was generated using [Nx](https://nx.dev).

### Quickstart

```bash
yarn
yarn nx serve --parallel --all
```

### Running an app

```bash
yarn nx run protocol:serve
```

### Building an app

```bash
yarn nx run protocol:build
```
